# Single-cell-transcriptomic-reveals-dual-and-multi-transmitter-use-in-neurons-across-metazoans
Code Accessibility

Here we list the codes used for the analysis of existing scRNAseq datasets to determine neurotransmitters coexpression.

> CElegans_script.html : Caenorhabditis elegans

> Ciona_script.html : Ciona intestinallis

> Drsophila_script.html : Drosophila melanogaster

> Hydra_script.html : Hydra vulgaris

> MMusculus_Adult_young_old_script.html : Mus musculus

> Schmidtea_script.html : Schmidtea mediterranea

> TurtleLizard_script.html : Trachemys scripta and Pogona vitticeps

> Zebrafish_script.html : Danio rerio
